POC Bot Fiscal

Assistente Fiscal via WhatsApp

Configure o Webhook, envie uma mensagem do número de teste e acompanhe o andamento das conversas e obrigações diretamente pelo painel.

Abrir Dashboard
Configuração Rápida
  1. Defina o VERIFY_TOKEN e gere um token temporário.
  2. Cadastre o webhook apontando para APP_BASE_URL/webhook.
  3. Adicione os testers no WhatsApp Cloud API e use os números de seed.
Fluxos Disponíveis

O bot cobre dois caminhos com state machine:

  • Simples Nacional — comércio com movimento.
  • Lucro Presumido — comércio com movimento.
Comandos Úteis

Durante a conversa, experimente enviar:

  • novo, nova empresa, encerrar
  • pendencias, resumo, problema
  • observacao, desabafo, pular, retomar
